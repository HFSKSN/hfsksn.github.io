## Messages

Angle Adjustment

|               | Byte 1-2 | Byte 3   | Byte 4   |
|---------------|----------|----------|----------|
| Variable Name | uint8_t  | uint16_t | uint16_t |
| Variable Type | uint8_t  | uint16_t | uint16_t |
| Min Value     | 8        | 0        | 0        |
| Max Value     | 8        | 360      | 360      |
| Example       | 8        | 350      | 25       |
